## Deskripsi
SIAKAD adalah aplikasi Sistem Informasi Akademik berbasis web yang dikembangkan untuk mengelola proses administrasi akademik secara terintegrasi. Sistem ini memungkinkan pengelolaan data mahasiswa, dosen, mata kuliah, nilai, dan KRS dengan lebih efisien.

## Fitur Utama
- Autentikasi dan manajemen pengguna
- Dashboard admin yang responsif
- CRUD Data Mahasiswa & Dosen
- Manajemen Mata Kuliah dan Nilai
- Pengelolaan KRS (Kartu Rencana Studi)
- Tampilan modern menggunakan AdminLTE + Tailwind CSS
